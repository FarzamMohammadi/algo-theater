## About

**algo-theater** is a free, open-source visual learning platform for algorithms and data structures. Built for visual learners who understand best when they can *see* concepts in action.

No dry textbook explanations. No static diagrams. Just clean, animated visualizations that map directly to how these algorithms actually work.

---

## Why This Exists

Learning algorithms from code alone is hard. You read the logic, you trace through it mentally, but something doesn't *click*. 

For visual learners, seeing is understanding. Watching a sorting algorithm swap elements in real-time, or seeing Dijkstra's algorithm flood across a graph â€” that's when it finally makes sense.

**algo-theater** brings every major algorithm to the stage, one visualization at a time.
